## 📝 文档规范

每个功能说明文档应包含以下内容：

1. **功能概述**：功能的核心特性和业务流程
2. **技术架构**：技术栈、系统架构图
3. **数据库设计**：相关数据表结构说明
4. **核心功能实现**：关键代码实现和逻辑说明
5. **配置说明**：外部服务配置（如有）
6. **使用说明**：详细的操作指南
7. **常见问题**：FAQ 和问题解决方案
