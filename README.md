# U3W 优立方 AI 主机：人机协同数智驱动的团队未来

版本：V2.75G

文档更新日期：2025年11月10日


U3W 优立方 AI 平台是一个集成多种 AI 智能体的综合内容创作平台，支持多端访问（PC 端、企业微信应用、微信小程序端、公众号后台），提供 AI 对话、内容生成、智能评分、媒体投递等功能。

#### 核心功能

•多 AI 智能体集成 - 在用户将自己的豆包、腾讯元宝、百度 AI、DeepSeek、秘塔和知乎直答绑定到平台后，支持将同一提示词向多个AI发问和追问。
•智能内容创作 - 支持文本生成、代码生成、图表生成等多种内容形式
•智能评分系统 - 对 AI 生成内容进行多维度评估，内置十几个预定义提示词，允许用户对提示词进行增删改。支持用户使用豆包或DeepSeek作为智能评分用的AI工具。
•媒体自动投递 - 支持投递到公众号草稿箱。
•多端协同 - PC 端、企业微信、小程序、公众号智能体数据同步，无缝协作。

#### 技术特色
•模块化架构 - 前后端分离，微服务架构设计
•实时通信 - WebSocket 技术实现实时 AI 对话
•智能提示词 - 支持自定义智能评分提示词模板，提升创作效率

#### 运行环境要求

- [JDK 17](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html) - Oracle JDK 17 官方下载

#### 首次部署主机引擎，仅需四步

1. **下载 [NB-U3W.jar](https://u3w.com/chatfile/NB-U3W.jar)**

```bash
java -jar NB-U3W.jar
```

2. **配置连接参数**

启动后按提示输入以下信息：

```bash
请输入地址：u3w.com/cubeServer
是否启用HTTPS/WSS？(y/n，默认n)：y
✅ 检测到可用端口：8083
请输入主机引擎ID：[您申请的主机ID]
```

3. **获取主机引擎 ID**
   - 联系管理员申请您的专属主机引擎 ID
   - 主机引擎 ID 用于绑定和识别您的主机实例

   **管理员联系方式：**

   <p align="center">
     <img src="docs-img/二维码.jpg" alt="管理员二维码" width="200">
   </p>

   <p align="center">扫描二维码联系管理员申请主机引擎 ID</p>

4. **开始登录使用**
   - 访问 [u3w.com](https://u3w.com)
   - 通过两次扫码，注册并登录系统
   - 在个人中心添加您管理员返回给您的主机引擎 ID
   - 开始体验优立方 AI 主机的强大功能！
